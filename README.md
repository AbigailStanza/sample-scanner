# 시료확인스캐너

엑셀 재고 목록과 라벨 QR 코드를 모바일에서 대조하는 **오프라인 PWA**.
실험실 시료 검수 시 목록-실물 불일치·누락 등 휴먼 에러를 방지한다.

> 🔗 **배포 주소:** https://abigailstanza.github.io/sample-scanner/
> 아이폰 **Safari**로 접속 → 공유 ▸ **홈 화면에 추가**로 PWA 설치(카메라·오프라인 동작).

## 주요 기능
- 📥 **엑셀 업로드 & 파싱** — 시료관리대장(.xlsx) 업로드, 시료번호 앞 9자리(`260112227-001` → `260112227`)를 고유 식별자로 추출, 날짜별 분류 저장
- 📋 **대화형 리스트** — 원본과 유사한 표, 행별 확인 체크박스·비고 입력, 시료번호/확인/비고 열 고정, 확대/축소
- 📷 **QR 스캔 대조** — 카메라로 라벨 스캔 → 목록 대조 후 상태 분기
  - 최초 확인: 자동 체크 + 효과음(녹색)
  - 중복 스캔: "이미 확인된 시료입니다"(주황)
  - 목록 없음: 붉은 점멸 + "해당 목록에 없습니다"(빨강)
- ⌨️ **수동 검색 fallback** — 라벨 훼손 시 9자리 직접 입력 조회·체크
- 💾 **자동 임시 저장** — 체크/비고를 IndexedDB에 실시간 저장, 새로고침·재실행 시 복구
- 📤 **엑셀 내보내기** — 원본 레이아웃 유지 + 확인(✓)·비고 칸 채워 다운로드
- 🌓 라이트/다크 자동, 장갑 착용 고려 터치영역 확대, 시료번호 monospace 강조

## 파일 구조
```
index.html              # 단일 페이지(스캔/리스트 탭)
style.css               # 라이트·다크, 터치, 고정열, 상태 색상
app.js                  # 파싱·렌더·스캔분기·검색·효과음·내보내기
db.js                   # IndexedDB 래퍼(데이터셋 저장/복구)
sw.js                   # 서비스워커(오프라인 캐시)
manifest.webmanifest    # PWA 매니페스트
icons/                  # 홈화면 아이콘
lib/xlsx.full.min.js    # SheetJS (로컬 포함)
lib/html5-qrcode.min.js # QR 스캐너 (로컬 포함)
```

## 로컬 실행 (PC 테스트)
정적 서버가 필요하다(서비스워커·모듈 동작을 위해 `file://` 직접 열기는 불가).
```bash
python -m http.server 8000
# 또는: npx serve .
```
브라우저에서 `http://localhost:8000` 접속.
> ⚠️ **카메라 스캔은 HTTPS(또는 localhost)에서만 동작.** PC 웹캠은 localhost에서 테스트 가능, 실제 폰 테스트는 아래 HTTPS 배포 후 진행.

## 배포 (iOS 사용 — GitHub Pages)
1. GitHub 저장소 생성 후 이 폴더의 파일 전체 push
2. 저장소 **Settings ▸ Pages** ▸ Source를 `main` 브랜치(루트)로 지정
3. 발급된 `https://<아이디>.github.io/<저장소>` 주소를 아이폰 **Safari**로 접속
4. 공유 ▸ **홈 화면에 추가** → PWA로 설치(오프라인 동작·전체화면)
5. 첫 스캔 시 카메라 권한 허용

> HTTPS가 자동 제공되어 iOS 카메라·PWA 설치·오프라인 캐시가 모두 동작한다.

## 제약사항
- 1회 업로드 최대 **1,000행** (초과분은 생략하고 경고)
- iOS Safari 최적화. QR 코드에는 9자리 시료번호가 담겨 있다고 가정(정규식 `\d{9}`로 유연 추출)
- SheetJS 커뮤니티판 특성상 내보내기 시 셀 값·구조는 보존되나 일부 서식/색상은 단순화될 수 있음
